3. Выберите правильный способ соединения (конкатенации) двух списков list1 и list2:

```python 
list1 = [‘a’, ‘b’, ‘c’, ‘d’]
list2 = [‘e’, ‘f’, ‘g’]
```

Результатом должен быть список:

```python 
list3 = [‘a’, ‘b’, ‘c’, ‘d’, ‘e’, ‘f’, ‘g’]
```

**Ответ**:

```python 
list3 = list1 + list2
```
